ServiceMax Inspector — Privacy Policy

Effective date: May 28, 2026 Version: 1.0.0

Short version. ServiceMax Inspector runs entirely in your browser. It does not collect, transmit, share, or sell any personal data. The only network calls it makes are to the Salesforce org you are already logged into.

1. Who this policy applies to

This policy describes how the ServiceMax Inspector Chrome extension (the “extension”) handles information when you use it. It applies to all users of the extension and to any version published to the Chrome Web Store.

2. What the extension does

ServiceMax Inspector is a productivity toolkit for ServiceMax administrators and developers working on the Salesforce platform. When you click the extension on a Salesforce page, it lets you:

  • Inspect the record you are viewing (including hidden and formula fields).
  • Run SOQL queries against your org and export the results.
  • Bulk insert, update, upsert, or delete records from a CSV.
  • Retrieve metadata, explore the REST and Tooling APIs, and view org limits.
  • Use ServiceMax-specific inspectors (Profile Tester, Find Test Records, and related tooling).

All of these actions run against the Salesforce org you are already authenticated to. The extension performs them on your behalf, using the same authorization you would have if you were clicking through the Salesforce UI yourself.

3. What information the extension reads

To do its job, the extension reads the following information from your browser. None of it is stored on, or transmitted to, any server controlled by the extension authors.

What is readWhyWhere it is sent
The Salesforce session cookie (sid) for the Salesforce domain in your current tab. To authenticate API calls back to your own Salesforce org. Only to your own Salesforce org, in the standard Authorization header. Never to any third party.
The current page URL and record identifier (e.g. the Id embedded in the URL). To scope inspection to the record you are viewing. Stays in the browser. Not transmitted off-device.
Any SOQL query you type, or CSV you paste, while using the extension. To execute the query or import you requested. Only to your own Salesforce org's APIs.

4. What the extension does not do

  • It does not collect, log, sell, share, or transfer any personal information.
  • It does not contact any server other than the Salesforce org you are logged into.
  • It does not include analytics, tracking pixels, advertising SDKs, or telemetry of any kind.
  • It does not load remote code. All JavaScript that runs is bundled in the extension package shipped through the Chrome Web Store.
  • It does not read pages outside the Salesforce domains listed in the manifest.

5. Permissions requested and why

cookies

Used solely to read the active Salesforce session cookie (sid) so the extension can authenticate API calls back to the same Salesforce org. The cookie value never leaves your browser except in the Authorization header of those calls to Salesforce itself.

Host permissions

The extension is only active on Salesforce-owned domains. Salesforce serves API and UI traffic across several domains depending on org type and cloud (commercial, U.S. Government Cloud, and the Microsoft Defender for Cloud Apps proxy), so each pattern must be permitted:

  • *.salesforce.com
  • *.force.com
  • *.cloudforce.com
  • *.visualforce.com
  • *.salesforce.mil
  • *.force.mil
  • *.cloudforce.mil
  • *.visualforce.mil
  • *.crmforce.mil
  • *.force.com.mcas.ms

The extension does not access, read, or modify any other website.

Content scripts

A small script is injected into Salesforce pages to add a launcher button so you can open the extension from any record. The script also reads the page URL to identify the record you are viewing. It does not modify Salesforce data and does not communicate with any third-party site.

6. Storage

The extension may store small amounts of non-personal preference data (such as your last-used query, column order, or import settings) in the browser's local extension storage. This data stays on your device and is never transmitted off it. You can clear it at any time by removing the extension from Chrome.

7. Third parties

The extension does not share any data with third parties. It does not integrate with any analytics, advertising, or data-broker service.

8. Children

The extension is a developer/admin tool intended for use by Salesforce administrators and developers in a workplace context. It is not directed at children under 13, and it does not knowingly collect information from anyone.
